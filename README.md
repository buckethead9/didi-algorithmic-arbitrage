# 🚀 DiDi Algorithmic Arbitrage: Optimization & Audit Pipeline

Este proyecto actúa como un **amplificador cognitivo** para auditar y optimizar la asimetría logística en plataformas de movilidad. El núcleo del sistema es la detección de la brecha entre la "Verdad Física" (Google Maps) y la "Percepción Algorítmica" (DiDi App).

## 🧠 Tesis Técnica: El Método de Arbitraje

El sistema no solo registra datos; audita la asimetría. La lógica implementada en SQL y Python clasifica los servicios según su potencial de rentabilidad:

1. **Arbitraje Agresivo (RO ≥ 1.5):** Servicios donde el algoritmo de la App otorga una ventaja competitiva superior al 50% frente a la distancia real.
2. **Optimización Estándar (RO ≥ 1.2):** Operación en rangos de eficiencia aceptables.
