# La Primera Huella
Registro de un proceso formativo que comienza sin garantías.

## Punto cero
Este repositorio marca el inicio documentado de un proceso de formación autónoma en análisis de datos.
No representa dominio técnico ni resultados, sino la decisión consciente de registrar el proceso desde su inicio.

## Fricción inicial
El primer contacto con Git y GitHub evidenció desconocimiento operativo, errores de ejecución y pausas deliberadas
para comprender el sentido de cada acción antes de continuar.

## Decisión metodológica
A partir de este punto, el proceso se documentará mediante control de versiones,
registrando decisiones relevantes —acertadas o no— como evidencia del aprendizaje real.
