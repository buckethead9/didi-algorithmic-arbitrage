# Processed Data

## Descripción
Esta carpeta contiene versiones limpias y/o enriquecidas del dataset original.

## Transformaciones aplicadas
- Validación de formatos numéricos.
- Normalización de unidades (km, COP).
- Cálculo de métricas derivadas (RO, utilidad_neta, salario_efectivo_hora, etc.).

## Uso
Archivos listos para:
- Carga en MySQL
- Análisis SQL
- Visualización
