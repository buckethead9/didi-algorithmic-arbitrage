# Raw Data

## Descripción
Esta carpeta contiene los datos originales recolectados manualmente durante la operación de DiDi Food.

## Características
- Datos sin transformaciones.
- Sin métricas derivadas.
- Formato CSV exportado desde Excel.

## Periodo
Diciembre 2025 - Enero 2026 (25 días operativos)

## Notas
- No contiene información personal de clientes.
- Datos anonimizados.
- Sirve como fuente base para el procesamiento en SQL.
